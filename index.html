OneDrive / SharePoint Auto-Sync LIVE

Connect your OneDrive or SharePoint Excel file — the dashboard pulls the latest data on demand. No drag & drop needed.

One-time setup:
1. Save your Excel to OneDrive or SharePoint
2. Open the file online → click Share → copy the link
3. Change the end of the URL: replace ?web=1 with ?download=1
4. Paste that download URL into ONEDRIVE_URL in this file's JS
5. Upload the updated index.html to GitHub once
After setup: Just update your Excel on OneDrive as usual, then click OneDrive Sync — the dashboard refreshes instantly with the latest data. Enable Auto-refresh and it updates itself every 5 minutes automatically.
